<p align="center">
  Android app indexing Termux tools via a metadata-driven pipeline.
</p>
<p>
  Termux Hub catalogs Termux tools with a reproducible, transparent workflow, storing validated data locally and updating stars and READMEs daily at 00:00 UTC.
</p>
<h2>How it works</h2>
<p>
  Fetches static GitHub metadata, processes tools, stars, and READMEs independently, stores them in a local Room database, and falls back to cached or bundled data if syncing fails.
</p>
<h2>Data sources</h2>
<p>
  All read-only data comes from <a href="https://github.com/maazm7d/TermuxHub">TermuxHub GitHub repo</a>, including tool metadata, stars, READMEs, thumbnails, and Hall of Fame entries.
</p>
<h2>Contributing</h2>
<p>
  To add or index a tool, see <a href="https://github.com/maazm7d/TermuxHub/blob/main/CONTRIBUTING.md">CONTRIBUTING.md</a>.
</p>
